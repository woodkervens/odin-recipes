# Site Web de Recettes — HTML

## Description du projet

Ce projet consiste à créer un site web de recettes de cuisine en utilisant **uniquement HTML**, sans CSS ni JavaScript. L'objectif est de structurer le contenu d'une page web de façon claire, sémantique et accessible, en s'appuyant sur les balises natives du langage pour organiser l'information (titres, listes d'ingrédients, étapes de préparation, images, liens, etc.).

Le site présente une ou plusieurs recettes, avec pour chacune :
- Un titre et une brève description
- Une liste des ingrédients
- Les étapes de préparation, numérotées
- Une image illustrant le plat

## Compétences démontrées

À l'issue de ce projet, les compétences suivantes auront été mises en pratique :

- **Organisation du contenu** : listes ordonnées (`<ol>`) pour les étapes, listes non ordonnées (`<ul>`) pour les ingrédients
- **Gestion des médias** : intégration d'images avec `<img>` et attributs `alt` appropriés
- **Accessibilité** : hiérarchie logique des titres (`<h1>` à `<h6>`), attributs `alt`, structure navigable
- **Liens et navigation** : création de liens internes et/ou externes avec `<a>`
- **Bonnes pratiques HTML** : validation du code, indentation cohérente, respect des standards du W3C

## Technologies utilisées

- HTML5 uniquement (aucun CSS, aucun JavaScript)
